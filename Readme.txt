Prerequisites
Ensure the following before proceeding:

Windows OS
Internet connection
Python installed (required for Flask)
Installation Steps
Step 1: Install Required Libraries
Locate the Install Libraries.bat file.
Double-click on the Install Libraries.bat file.
If a prompt appears, allow the action to proceed.
Wait for the libraries to install. This process may take a few minutes.
Step 2: Launch SentryLabs (Flask Application)
Open a terminal (Command Prompt or PowerShell).
Navigate to the directory where the SentryLabs project is located.
Run the following command to start the Flask application:
bash
Copy code
flask run
After starting the Flask server, a browser window will automatically open with the SentryLabs interface.
Follow any prompts that appear within the application.
Optional Steps
Step 3: Update Datasets (Optional)
If the datasets have been updated:

Navigate to the Side Navigation (Side Nav) menu.
Press the Update button to refresh the datasets.
Step 4: Train Models (Optional)
To train the models:

Go to the Side Navigation (Side Nav) menu.
Press the Train button to initiate the training process.
