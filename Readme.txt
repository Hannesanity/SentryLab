Installation Steps
Open Install Libraries.bat
Double-click on the Install Libraries.bat file.
If a prompt appears, allow the action to proceed.
Wait for Libraries to Install
The batch file will automatically install the required libraries. This process may take a few minutes.
Open SentryLabs
Once the libraries are installed, open the SentryLabs application.
Check the Prompts
Follow any prompts that appear within the application.
The browser will open automatically after completing the prompts.
Optional Steps
Update Datasets
If the datasets have been updated, navigate to the Side Navigation (Side Nav) menu.
Press the Update button to refresh the datasets.
Train Models
To train the models, go to the Side Navigation (Side Nav) menu.
Press the Train button to initiate the training process.
